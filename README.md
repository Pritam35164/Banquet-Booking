# BanquetBook – Streamlined Banquet Hall Booking Platform

BanquetBook is an elegant and user-friendly web platform designed to simplify event planning. It enables clients to search, explore, and book luxurious banquet halls across India, while venue owners can manage listings, bookings, and payments seamlessly—all within the same system.

---

## Key Features

### For Event Clients
- 🔍 **Smart Search** – Filter venues by city, event date, and guest count
- 🏛 **Venue Details Page** – View stunning images, amenities, prices, and capacity
- 📅 **Real-Time Availability** with visual indicators
- 📂 **Client Dashboard** – Track bookings, payments, and event details
- 💳 **Simulated Secure Payments** – Multiple payment modes (Card, UPI, Net Banking)
- 🧭 **Responsive & Intuitive UI** – Optimized for all devices

### For Venue Owners
- 🏷 **Add & Manage Venues** – Upload details, images, and availability
- ✅ **Booking Requests Panel** – Accept or reject client bookings
- 📆 **Availability Calendar Management**
- 💰 **Track Payments & Booking Status**
- 📊 **Dashboard Insights** (future enhancement)
